# SAE-1.04-A1 - Fripkil

Projet réalisé dans le cadre de la SAE 1.04. Application de gestion pour une friperie (Fripkil).

## Équipe

*   **LAHURTE Noé** - Gestion des Achats
*   **BIZEL-BIZELLOT Léandre** - Gestion des Achats de Vêtements
*   **ALLALI Yanis** - Gestion des Déposes
*   **BONVALLOT Mathis** - Gestion des Collectes de Vêtements

## Fonctionnalités

L'application est divisée en plusieurs modules gérés par les membres de l'équipe :

### Achat (Noé)
*   Visualisation des achats (`/Achat/show`)
*   Statistiques des achats (`/Achat/etat`)

### Achat-Vêtements (Léandre)
*   Gestion des détails d'achats de vêtements (`/Achats-vetements/show`)
*   Statistiques sur les types de vêtements achetés (`/Achats-vetements/etat`)

### Collecte-Vêtements (Mathis)
*   Gestion des catégories et collectes (`/Collecte-vetements/show`)
*   Statistiques sur les collectes (`/Collecte-vetements/etat`)

### Dépose (Yanis)
*   Gestion des dépôts clients (`/Depose/show`)
*   Statistiques des déposes (`/Depose/etat`)
